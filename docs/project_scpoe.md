# CancerAI Project Scope

## Project Title

CancerAI: Explainable Multi-Cancer Classification using RNA-Seq Gene Expression Data

---

## Objective

Develop an end-to-end machine learning platform that predicts cancer types using RNA-Seq gene expression data and provides explainable predictions.

---

## Supported Cancer Types

- BRCA (Breast Cancer)
- COAD (Colon Adenocarcinoma)
- KIRC (Kidney Renal Clear Cell Carcinoma)
- LUAD (Lung Adenocarcinoma)
- PRAD (Prostate Adenocarcinoma)

---

## Technologies

- Python
- Scikit-learn
- XGBoost
- SHAP
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Joblib

---

## Planned Features

- Data Validation
- Exploratory Data Analysis
- Data Preprocessing
- Feature Selection
- Machine Learning Models
- Explainable AI (SHAP)
- Interactive Streamlit Web Application
- Prediction Report Generation
