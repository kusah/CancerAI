# CancerAI v1.0 Scope

## Goal

Build an explainable machine learning platform for multi-cancer classification using TCGA-derived RNA-Seq gene expression data.

## Supported Cancer Types

- BRCA
- LUAD
- COAD
- KIRC
- LIHC

## Features

- Data preprocessing
- Exploratory Data Analysis
- Feature Selection
- Baseline ML Models
- XGBoost
- SHAP Explainability
- Gene Explorer
- Streamlit Deployment
